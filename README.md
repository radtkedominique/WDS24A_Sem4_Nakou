# Lebensmittelpreise – Preisprognose mit ML
Kann ein ML-Modell die zukünftige Preisentwicklung von Lebensmittelkategorien vorhersagen? Zusätzlich wird untersucht, ob sich Produkte anhand ihres Preisverhaltens in Risikogruppen (stark inflationsbetroffen vs. stabil) klassifizieren lassen – mit direktem Nutzen für Verbraucher und Einkaufsplanung.
